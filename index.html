Prof. Dr. Tegawendé F. BISSYANDE

Chief Scientist (Professor category) with SnT at University of Luxembourg
ERC Fellow - PI of NATURAL (Artificial Intelligence for Program Repair)
bissyande @Google Scholar @DBLP @Orbi.lu

Research interests:

  • Software Debugging (especially bug localization and program repair ),
  • Software Security (especially malware detection and analysis),
  • Code Search (both free-form and semantic code-to-code),
  • Machine Learning and Natural Language Processing for software engineering,
  • and more broadly software engineering and cyber-security.
